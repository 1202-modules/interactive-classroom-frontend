# Документация Frontend

Frontend приложение для платформы интерактивных занятий Interactive Classroom Platform.

## Описание

Клиентское веб-приложение (SPA) для платформы интерактивных занятий. Предоставляет интерфейсы для трех ролей: преподаватель, студент и проектор. Студенты подключаются к сессии через QR-код, сканируя его смартфоном.

## Основные функции

- Интерфейс преподавателя: создание сессий, управление интерактивными режимами, модерация
- Интерфейс студента: подключение по QR-коду, участие в Q&A, викторинах, чате, отправка реакций
- Интерфейс проектора: отображение вопросов, результатов опросов, реакций аудитории на большом экране
- Real-time обновления через WebSocket
- Адаптивный дизайн для мобильных устройств
- Поддержка мультиязычности интерфейса
- Встраивание презентаций (PDF, PPTX)

## Технологии

- JavaScript/TypeScript
- Современный веб-фреймворк
- WebSocket для real-time коммуникации
- Адаптивная верстка
