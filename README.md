# Interactive Classroom Frontend

> [Русская версия](docs/README-RU.md)

Frontend application for the Interactive Classroom Platform - a web application for conducting interactive classes in university auditoriums.

## Overview

Client-side web application (SPA) for the interactive classroom platform. Provides interfaces for three roles: teacher, student, and projector. Students connect to sessions via QR code using their smartphones.

## Key Features

- Teacher interface: session creation, interactive mode management, moderation
- Student interface: QR code connection, participation in Q&A, quizzes, chat, sending reactions
- Projector interface: displaying questions, poll results, audience reactions on a large screen
- Real-time updates via WebSocket
- Responsive design for mobile devices
- Multi-language interface support
- Presentation embedding (PDF, PPTX)
